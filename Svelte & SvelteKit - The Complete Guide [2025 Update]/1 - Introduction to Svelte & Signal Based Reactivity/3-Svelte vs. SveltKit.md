# Svelte vs. SvelteKit

### Svelt
* Componente baseado em JavaScript
* Utilizado para construir componentes web
### SvelteKit
* É um framework para construir aplicações web de extrema alta performance
* Utiliza-se roteamento, offline support, SSR, progressão e mais

> SvelteKit para Svelte é como NextJS para ReactJS ou NuxtJS para Vue

Podemos criar uma aplicação utilizando apenas Svelte, mas é muito improvável que façamos isso 😅
